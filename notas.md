## Flujo basico git
- git add .

## Comandos git
- git init //Creo repositorio
- git add . //Agrega todos los archivos al commit.
- git remote add origin(URL del repositorio) // Conectar proyecto con GitHub.
- git branch -M main
- git push -u origin main //Subir proyecto.
- git status //Estado del proyecto, muestra archivos que git no sigue aun.
- git log //Visualizar los commits realizados.
